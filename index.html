About Me

I graduated with a BS in Computer Science from CUNY City College Of New York with a passion for coding I bet you hear that a lot. My favorites language is Java; however, I have used other languages. I've used Python (web development), Javascript (web development, machine learning), and C++. I am currently teaching myself the Spring framework, along with Spring Sercuity, Hibenrate, JSP, and MySQL. I am currently working on a web application using Spring, this web application is an online forum, where users can create threads and discuss topics they're interested in. Along side with this project I am working on a graph visualization project that will help users visualize how graph search algorithms work. This project is implemented using Javascript. I hope to teach myself game development since video games were one of the big reasons why I majored in Computer Science. I said at the beginning my passion is coding, but where my passion truely lies is creating things.
